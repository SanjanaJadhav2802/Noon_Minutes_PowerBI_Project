## Executive Summary

This project presents an end-to-end Power BI analytics dashboard for *Noon Minutes*, a quick-commerce platform in Dubai operating on a 15-minute delivery promise.

The objective of this analysis is to evaluate:

- Sales performance trends
- Delivery efficiency
- Cancellation patterns
- Outlet-level performance
- Item-level revenue contribution

The dashboard provides actionable insights to improve operational efficiency, reduce cancellations, and support strategic outlet expansion decisions.

---

## Business Problem

Noon Minutes is experiencing fluctuations in:

- Sales performance
- Cancellation rates
- Delivery time efficiency

The business needs data-driven answers to:

1. Why are sales declining in certain locations?
2. Does increased delivery time lead to higher cancellation rates?
3. Which outlet size and location generate the highest revenue?
4. Which item categories drive maximum sales?
5. How does order status impact delivery time performance?

---

## Solution Approach

The solution was developed using Power BI, applying data modeling, DAX measures, and visual analytics.

### 1. Sales Performance Analysis

- Total Sales
- Total Orders
- Average Order Value (AOV)
- Average Sales per Order
- Sales Trend Over Time

### 2. Delivery Performance Analysis

- Average Delivery Time
- Delivery Time by Order Status
- Cancellation Rate (%)
- Delivery Time Distribution (Time Bins)
- Trend Analysis by Date

### 3. Outlet Performance Analysis

- Sales by Outlet Location
- Sales by Outlet Size
- Order Volume by Outlet
- Delivery Efficiency by Outlet

### 4. Item-Level Insights

- Sales by Item Type
- Orders by Item Category
- AOV by Item Category
